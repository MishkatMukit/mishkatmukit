## Enabling the Tetris Graph

The Tetris animation above is powered by the [`contribution-graph-art`](https://github.com/leereilly/contribution-graph-art) GitHub Action. It runs daily, reads your real contribution data, and publishes an animated SVG to an `output` branch. Two files are needed in your `MishkatMukit/MishkatMukit` repo:

**1. `.github/workflows/contribution-graph-art.yml`**

```yaml
name: Generate Contribution Art

on:
  schedule:
    - cron: "0 0 * * *" # daily at midnight UTC
  workflow_dispatch:
  push:
    branches: [main]

jobs:
  generate:
    runs-on: ubuntu-latest
    permissions:
      contents: write
    steps:
      - uses: actions/checkout@v4

      - name: Generate graph with falling tetrominoes
        uses: leereilly/contribution-graph-art@v1
        with:
          github_user_name: ${{ github.repository_owner }}
          tetromino_count: "5"
          palette: "auto"
          output_path: dist/contribution-graph-tetrominoes.svg

      - name: Push to output branch
        uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

**2.** Commit and push this file to your `main` branch. The first run publishes `contribution-graph-tetrominoes.svg` to an `output` branch, which is exactly what the `<picture>` block in this README already points to — no further edits needed once the action runs once.
